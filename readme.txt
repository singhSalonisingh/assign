What is the use of this Repo

This Project is a Simple ExpressJS with mongodb Project which demonstrates the following


Making HTTP calls
Communicating between parent and child component

Using Basic Routing 
The project Template can be used to build bigger projects


Prerequisites
Install Node JS
Refer to https://nodejs.org/en/ to install nodejs

Install Expressjs

Install mongoose

Install nodemon

download mongodb with mongodb compass


Install create-react-app
Install create-react-app npm package globally. This will help to easily run the project and also build the source 
files easily. Use the following command to install
create-react-app

npm install -g create-react-app


Click on the link to see the application

Cloning and Running the Application in local
Clone the project into local

Install all the npm packages. Go into the project folder and type the following command to install all npm packages

npm install
In order to run the application Type the following command

nodemon app
The Application Runs on localhost:5000
